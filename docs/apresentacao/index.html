Vagrantfile (Trecho LVM)

# Fixando Hardware
vb.memory = "4096"
vb.cpus = 2
vb.customize ['createhd', '--filename', './disk.vdi', '--size', 20480]

# Implantando LVM Sem Falha Humana
agt.vm.provision "shell", inline: <<-SHELL
  pvcreate /dev/sdb
  vgcreate vg_data /dev/sdb
  lvcreate -l 100%FREE -n lv_data vg_data
  mkfs.ext4 /dev/vg_data/lv_data
SHELL

SecOps & Hardening

Blindagem ao Nível do Kernel

O desafio não pedia segurança explícita, mas um Global Support Engineer Sênior jamais projeta ambientes abertos. Programei um Shell System para erradicar da fundação o maior vetor de ataque Linux: o serviço de SSH (Purga de Pacotes).

  • ✔ Purge do openssh-server local
  • ✔ Mitigação Sysctl p/ SYN Flood/IP Spoofing
  • ✔ Proteção anti Roteamento Malicioso em Rede
user_data.sh (Shell Hardening Exclusivo)

# Purga completa de acessos remotos tradicionais SSH
systemctl stop ssh || true
systemctl disable ssh || true
apt-get purge -y openssh-server

# Hardening Anti DDOs de Sysctl (SYN flood, IP Spoof)
cat << 'EOF' > /etc/sysctl.d/99-security.conf
net.ipv4.conf.all.accept_source_route = 0
net.ipv4.conf.all.accept_redirects = 0
net.ipv4.tcp_syncookies = 1
EOF

sysctl -p /etc/sysctl.d/99-security.conf
